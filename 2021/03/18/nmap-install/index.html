Nmap 安装简记

发表于2021-03-18|更新于2022-09-23|Security
|字数总计:460|阅读时长:1分钟|阅读量:

Nmap 官网及软件下载方式

Download the Free Nmap Security Scanner for Linux/Mac/Windows

访问链接稍微往下拉就可以看到图中标注的下载地址了。

同意使用协议

点击 I agree 进入下一步。

选择需要安装的组件

根据自己的需要来勾选需要安装的组件，比如 不需要图形界面则可将 Zenmap 取消勾选。

注：关于各组件的详细使用方法，请参阅官方文档或其他网络博客。

组件名 作用
Nmap Core Files Nmap 核心文件
Register Nmap Path 将 Nmap 写入系统路径（Path），以便在命令行中调用
Npcap 1.00 Nmap 依赖的网络探测工具，如已安装，则可以取消勾选
Network Performance Improvements 网络性能改进
Zenmap (GUI Frontend) ‎官方跨平台 Nmap 安全扫描仪 GUI‎
Ndiff (Scan comparison tool) ‎比较‎‎ Nmap‎‎ 扫描结果的实用工具‎
Ncat (Modern Netcat reincarnation) 官方对 Netcat （业内口中的“瑞士军刀”） 的重新实现
Nping (Packet generator) 网络数据包生成工具

设置安装目录

安装软件时，一般是放置在 非系统盘 (通常指 C盘) 的文件夹内，如 D:\Program Files (x86)\Nmap

到了这一步将会弹出 Npcap 的安装程序，根据指引操作即可。

安装 Npcap

同上，点击 I Agree

设置软件属性

这四个选项的中文含义分别为

  • 对 Nmap 7.80 和更早的版本进行支持
  • 将 Npcap 设置为 只有管理员可以访问（使用）
  • 支持无线网卡
  • 在 WinPcap API 兼容模式下安装 Npcap

注：此步骤通常默认即可

安装完成

点击 Finish，待 Nmap 安装完成。

创建快捷方式

  • 开始菜单文件夹
  • 桌面图标

使用测试

Windows 命令提示符下输入 nmap 即可返回帮助信息。
